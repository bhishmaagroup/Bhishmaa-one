# Bhishmaa One

Bhishmaa One is a production-grade Multi-Tenant Business Operating System SaaS platform.

It is designed to support:
- Billing & POS Invoicing (GST ready, thermal printing, QR codes)
- Inventory & Stock Management (barcodes, expiry tracking)
- Customers & Suppliers CRM
- Staff, Employee Attendance & Payroll Lite
- Financial reports & Auditing
- Modular Industry extensions (Coaching, Restaurants, Medical, Agencies)

## Stack
- Backend: Flask, SQLAlchemy, Blueprint, Celery, Redis
- Database: PostgreSQL (Primary), SQLite (Offline/Demo fallback)
- Frontend: Bootstrap 5, HTMX, Vanilla JS

## Launch
```powershell
pip install -r requirements.txt
python run.py
```
